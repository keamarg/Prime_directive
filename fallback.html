Prime Directive - Game Loading Issue

Error: CORS policy prevented loading game data files

This happens when you try to open the game directly from your file system instead of using a web server.

How to Fix This

To play the game properly, you need to run it through a local web server. Here are the steps:

  1. Open your terminal/command prompt
  2. Navigate to the game directory: 
    cd /path/to/Prime_directive
  3. Start a simple Python web server: 
    python -m http.server 8000
    or 
    python3 -m http.server 8000
  4. Open your browser and go to: 
    http://localhost:8000

The game needs to be accessed through a web server because modern browsers enforce security policies that restrict loading data from local files.

Alternative Solutions

If you can't run a Python server, you can:

  • Use any other local web server (Node.js, PHP, etc.)
  • Install a browser extension that disables CORS (not recommended for security reasons)
  • Edit the game code to load embedded data instead of external JSON files
